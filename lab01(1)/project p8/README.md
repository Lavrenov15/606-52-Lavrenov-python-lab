Задание:
Написать игру крестики-нолики с графическим интерфейсом (GUI) на библиотеке Flet

Ход выполнения:
1. Создание класса Player (Игрок)
Создаётся класс Player с атрибутами: symbol (символ игрока 'x' или 'o'), name (имя игрока), moves (строка с занятыми позициями)
Метод add_move(position) добавляет номер позиции в строку moves игрока
Метод check_win() проверяет, есть ли в moves игрока хотя бы одна из 8 выигрышных комбинаций ("123", "456", "789", "147", "258", "369", "159", "357")

2. Инициализация игры (метод __init__ класса TicTacToe)
Создаётся главное окно приложения page с заголовком "Крестики-нолики"
Создаются два игрока: player_x (Крестики) и player_o (Нолики)
Устанавливается текущий игрок current_player = player_x (первыми ходят крестики)
Создаётся список доступных позиций available_positions = [1, 2, 3, 4, 5, 6, 7, 8, 9]
Создаётся пустой словарь buttons для хранения кнопок игрового поля
Устанавливается флаг game_active = True (игра активна)
Вызывается метод create_interface() для создания графического интерфейса

3. Создание графического интерфейса (метод create_interface)
Создаётся заголовок игры title_text с текстом "Крестики-нолики"
Создаётся текст статуса status_text с информацией о текущем игроке ("Ход: Крестики (X)")
Создаётся игровое поле в виде сетки 3×3 с помощью вложенных циклов:
Внешний цикл for i in range(3) создаёт 3 строки (Row)
Внутренний цикл for j in range(3) создаёт 3 кнопки в каждой строке
Каждая кнопка получает номер позиции от 1 до 9
При нажатии на кнопку вызывается метод make_move(position)
Создаётся кнопка "Новая игра", при нажатии вызывается метод restart_game()
Все элементы добавляются на страницу page в вертикальную колонку (Column)

4. Обработка хода игрока (метод make_move)
Проверяется активна ли игра (if not self.game_active: return)
Проверяется свободна ли выбранная позиция (if position not in self.available_positions: return)
Позиция удаляется из списка доступных available_positions
Вызов метода add_move(position) у текущего игрока (добавляет позицию в его ходы)
Обновляется текст на кнопке: ставится символ текущего игрока ('X' или 'O')
Кнопка блокируется (button.disabled = True)
Обновляется интерфейс (self.page.update())

5. Проверка победы
Вызывается метод check_win() у текущего игрока
Если метод возвращает True (игрок собрал выигрышную комбинацию):
Флаг game_active устанавливается в False
Вызывается метод show_result() с сообщением "ПОБЕДА! Крестики/Нолики выиграли! 🎉"
Выход из функции (return)

6. Проверка ничьи
Проверяется, закончились ли доступные позиции (if not self.available_positions)
Если свободных позиций нет:
Флаг game_active устанавливается в False
Вызывается метод show_result() с сообщением "НИЧЬЯ! 🤝"
Выход из функции (return)

7. Смена игрока (метод switch_player)
Если игра не завершена, текущий игрок меняется на противоположного
Обновляется текст статуса: status_text.value = f"Ход: {новый_игрок.name} ({новый_игрок.symbol.upper()})"
Обновляется интерфейс

8. Отображение результата (метод show_result)
Текст статуса status_text меняется на сообщение о победе или ничье
Размер текста увеличивается до 24
Текст становится жирным (weight = ft.FontWeight.BOLD)
Обновляется интерфейс

9. Перезапуск игры (метод restart_game)
Флаг game_active устанавливается в True
Очищаются ходы у обоих игроков (moves = "")
Текущий игрок сбрасывается на крестики
Список доступных позиций заполняется заново (list(range(1, 10)))
Все кнопки очищаются (текст становится пустым) и разблокируются
Текст статуса возвращается к стандартному виду (размер 20, обычный шрифт)
Обновляется интерфейс

10. Запуск приложения (функция main)
Устанавливаются параметры окна: ширина 450, высота 600, изменение размера запрещено
Создаётся экземпляр игры TicTacToe(page)
Вызывается ft.app(target=main) для запуска приложения

11. Цикл игры
После каждого хода, если игра не завершена, управление возвращается к ожиданию следующего нажатия на кнопку
Игра продолжается до победы одного из игроков или наступления ничьей
После завершения игры можно нажать кнопку "Новая игра" и начать заново
